New Horizons Journey to Pluto

For my first major mobile-first, responsive interactive design project in MCAD's Responsive Design class with Robert Algeo, I designed and developed a website commerating the historic Pluto flyby by NASA's New Horizons spacecraft. In addition, this project introduced me to Style Tiles, a design method for rapid prototyping.

Visit site.

Style Tile

Project Goals

  • Mobile-first, responsive layout
  • Rapid design concepts with iterative Style Tiles
  • Image lightbox
  • Browser fall-back detection

Technologies

  • HTML5
  • CSS3, SASS
  • JavaScript, jQuery
